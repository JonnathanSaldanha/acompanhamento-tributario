Ajuda e Suporte
Como Usar

Importação de Dados

Para importar dados, vá para a aba "Importar Dados" e selecione um arquivo Excel com as colunas: Empresa, Valor, Data e Tipo.

Manipulação de Dados

Na aba "Manipulação de Dados", você pode criar operações personalizadas para somar, calcular médias ou agrupar dados conforme sua necessidade.

Configurações

Personalize o comportamento do sistema na aba "Configurações".

Dúvidas Frequentes

Quais formatos de arquivo são suportados?

O sistema suporta arquivos Excel (.xlsx e .xls).

Como remover dados duplicados?

Durante a importação, você pode escolher se deseja remover dados duplicados automaticamente ou mantê-los.

Posso exportar os resultados?

Sim, você pode exportar dados para Excel, PDF e imagem em várias partes do sistema.
